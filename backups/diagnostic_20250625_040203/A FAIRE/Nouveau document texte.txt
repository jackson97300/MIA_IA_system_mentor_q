📋 GUIDE ÉTAPE PAR ÉTAPE : CONFIGURATION DISCORD
🎯 OBJECTIF
Configurer Discord pour recevoir les notifications de votre bot de trading.
📝 ÉTAPE 1 : CRÉER VOS WEBHOOKS DISCORD
1.1 Ouvrir Discord

Lancez Discord (application desktop ou web)
Allez dans votre serveur (ou créez-en un pour le bot)

1.2 Créer les canaux
Créez 4 canaux texte :
📁 Votre Serveur Discord
  ├── 📢 #signals-live       
  ├── 💰 #trades-executed    
  ├── 🚨 #alerts-critical    
  └── 📊 #daily-reports
Comment faire :

Clic droit sur votre serveur → "Create Channel"
Type : Text Channel
Nommez-les comme ci-dessus

1.3 Créer les webhooks
Pour CHAQUE canal :

Cliquez sur le canal (ex: #signals-live)
Cliquez sur ⚙️ (settings du canal)
Menu gauche → "Integrations"
Cliquez "Webhooks"
Cliquez "New Webhook"
Donnez un nom (ex: "MIA Signals Bot")
Cliquez "Copy Webhook URL"
Gardez cette URL (notepad temporaire)

Répétez pour les 4 canaux !
📝 ÉTAPE 2 : CRÉER LE FICHIER .env
2.1 Copier le template
bash# Dans le terminal/cmd
cd D:\MIA_IA_system

# Copier le template
copy .env.example .env
OU manuellement :

Trouvez .env.example
Copiez-le
Renommez la copie en .env (sans le .example)

2.2 Ouvrir .env dans un éditeur
Ouvrez avec Notepad++ ou VSCode :
bashnotepad .env
📝 ÉTAPE 3 : REMPLACER LES WEBHOOKS
3.1 Ce que vous verrez dans .env
env# Canal pour les signaux détectés (tous les signaux)
DISCORD_WEBHOOK_SIGNALS=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN

# Canal pour les trades exécutés (trades pris seulement)
DISCORD_WEBHOOK_TRADES=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN

# ... etc
3.2 Remplacer par vos vrais webhooks
EXEMPLE CONCRET :
Si votre webhook pour #signals-live est :
https://discord.com/api/webhooks/1234567890/abcdefghijklmnop
Remplacez :
env# AVANT
DISCORD_WEBHOOK_SIGNALS=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN

# APRÈS
DISCORD_WEBHOOK_SIGNALS=https://discord.com/api/webhooks/1234567890/abcdefghijklmnop
Faites ça pour les 4 webhooks !
3.3 Fichier .env final
env# Configuration Discord pour MIA Trading Bot
DISCORD_ENABLED=true
DISCORD_BOT_NAME=MIA Bataille Navale Bot

# Webhooks (avec vos vraies URLs)
DISCORD_WEBHOOK_SIGNALS=https://discord.com/api/webhooks/1234567890/abcdefghijklmnop
DISCORD_WEBHOOK_TRADES=https://discord.com/api/webhooks/0987654321/qrstuvwxyz123456
DISCORD_WEBHOOK_ALERTS=https://discord.com/api/webhooks/1357924680/789456123abcdefg
DISCORD_WEBHOOK_PERFORMANCE=https://discord.com/api/webhooks/2468013579/hijklmnop7894561
3.4 Sauvegarder

Ctrl+S pour sauvegarder
Fermez l'éditeur

🔧 TROUBLESHOOTING
❌ "Webhook not found"

Vérifiez que l'URL est complète
Vérifiez pas d'espaces avant/après

❌ "Permission denied"

Le bot a-t-il accès au canal ?
Le webhook est-il actif ?

❌ ".env not found"
python# Ajoutez au début de votre script :
from dotenv import load_dotenv
load_dotenv()  # Charge le .env
